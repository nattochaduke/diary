#### 2.2.5 Projective measurements

量子情報ではprojective measurementsという公準3の特別な場合(実際にはユニタリ変換を考えれば同値, Sect. 2.2.8)を扱うことが多い.

Projective measurements projective measurementはobservableと呼ばれるエルミート作用素$M$で,

$$M = \sum_m m P_m$$

というスペクトル分解を持つ. $P_m$は$M$の固有値$m$に対応する固有空間への射影作用素である.

おさらい

スペクトル分解

エルミート行列$A$の固有値は実数であり,$\lambda_1 \leq \lambda_2 \leq \cdots \leq \lambda_n$とする.

対応する単位固有ベクトル$x_i$を$x_j^* x_i = \delta_{ij}$を満たすように選べて,

$$A(x_1 … x_n) = (x …, x_n) \text{diag}(\lambda_1 …., \lambda_n)$$

$(x_1 … x_n)$はユニタリ行列($\Leftrightarrow$列が正規直交基底をなす)だから,

$$A = (x_1 … x_n) \text{diag} (\lambda_1, …, \lambda_n) (x^*_1 , …, x^*_n)^T \ = \sum_i \lambda_i x_i x_i^*$$

が成立.

$P_i = x_i x^*_i \in M_n(\mathbb{C})$とすると

$$A = \sum_i \lambda_i P_i$$

これは

$$P_i^2 = P_i$$

$$P_i^* = P_i$$

$$P_i P_j = 0$$

を満たし,$A$のスペクトル分解という. (http://www.math.s.chiba-u.ac.jp/~nagisa/lecture04/fa3.pdf)

射影作用素

線形空間とその部分空間$W \subset V$があって,$dim(W) = k, dim(V) = d$とする.

$V$の正規直交基底$\ket 1, \ket 2, …, \ket d$で,

$\ket 1, \ket 2,…, \ket k$が$W$の正規直交基底でもあるものが存在する.

$$P = \sum_i^k \ket i \bra i$$

を$W$への射影作用素(projector)という. (2.1.6 Adjoints and Hermitian operators)

$\ket \psi$を観測するとき結果$m$を得る確率は $$p(m ) = \bra \psi P_m \ket \psi$$

観測によって変化した状態は

$$\frac{P_m \ket \psi}{\sqrt{p(m)}}$$

である.

projective measurementsには様々な良い性質がある.その一つ平均値の計算がかんたんなことである.

$$\begin{aligned}E(M) &= \sum_m mp(m) \ &=\sum_m m\bra \psi P_m \ket \psi \ &= \bra \psi \left(\sum_m mP_m \right) \ket \psi \ &= \bra \psi M \ket \psi\end{aligned}$$

$E(M) = $と略して書くこともある.

単に$\sum_m P_m = I, P_m P_{m’} = \delta_{mm’}P_m$なる$P_m$を並べることで$M=\sum_m mP_m$でprojective measurementを行うことを暗示したり,“measure in basis $\ket m$"で$P_m = \ket m \bra m$でprojective measurementを行うことを暗示したりする.

1-qubitの系でのprojective measurementの例を示す.

Pauliの$Z$ゲート

$$Z = \left(\begin{array}{} 1 & 0 \ 0 & -1 \end{array} \right)$$

をprojective measurementと考えると,これは固有値$+1$と$-1$を持ち,対応する固有ベクトルは$(1, 0)^T, (0, 1)^T$すなわち$\ket 0, \ket 1$である.

$\ket \psi = (\ket 0 + \ket 1) / \sqrt{2}$を観測すれば+1が$\braket{\psi}{0}\braket{0}{\psi} = 1 / 2$の確率で得られ,$-1$が同様に$1/2$の確率で偉える.

2.2.6 POVM measurements

POVMは’Positive Operator-Valued Measure’の略. 観測後の状態に興味がなく,観測結果かその統計量だけに興味があるときに使う. 公準3から,

$M_m$という観測が$\ket \psi$に作用したとき$m$が得られる確率は

$$p(m) = \bra \psi M^\dagger_m M_m\ket \psi$$

$$E_m = M^\dagger_m M_m$$

と定義すると,$E_m$は正値行列で$\sum_m E_m = I, p(m) = \bra \psi E_m \ket \psi$.

$E_m$をPOVM elementといい,${E_m}$をPOVMという.

Projective measurementでは$P_mP_{m’} = \delta_{mm’}P_m$だから$E_m = P_m$が成立.

2.4 The density operator

これまでは系のある状態をstate vector (状態ベクトル)として扱ってきたが,数学的には同値だが異なった表現の方法にdensity operatorやdensity matrixと呼ばれる方法がある.

2.4.1 Ensembles of quantum states

density operator (密度作用素)表現は状態が完全には判明していない系を表現するのに便利である. ある系がいくつかの${\ket {\psi_i}}$うちのどれか一つであり,その確率は${p_i}$とする.

${p_i, \ket {\psi_i}}$をensemble of pure statesと呼ぶ.

$$ \rho = \sum_i p_i \ket {\psi_i} \bra {\psi_i}$$

によってdensity operatorあるいはdensity matrix $\rho$を定める.

例えば閉じた系でのユニタリ作用素$U$を考える. 系の最初の状態が確率$p_i$で$\ket {\psi_i}$であるとき,$U$が適用されたあとの系の状態は確率$p_i$で$U\ket {\psi_i}$である.よって密度作用素の時間発展は

$$\rho = \sum_i p_i \ket{\psi_i} \bra{\psi_i} \rightarrow^U \sum_i p_i U\ket {\psi_i} \bra{\psi_i} U^{\dagger} = U\rho U^\dagger$$

となる. 観測作用素$M_m$も密度作用素表現によく馴染む.

$\ket {\psi_i}$を観測して$m$が得られる確率は

$$p(m|i) = \bra {\psi_i} M^\dagger_m M_m \ket{\psi_i} = \text{tr}(M^\dagger_m M_m \ket{\psi_i}\bra{\psi_i})$$

よって

$$p(m) = \sum_i p(m|i)p_i = \text{tr}(M^\dagger_m M_m \rho) $$
