2.2 The postulates of quantum mechanics

公準1

ある物理系に紐付けられた内積の入った複素ベクトル空間をその系のstate space状態空間という. その系はその状態空間の単位ベクトルであるstate vectorによって完全に記述できる.

公準2

閉じた系の時間発展はユニタリ変換であり,離散時間の時間発展はユニタリ行列$U$によって

$$\ket {\psi’} = U\ket \psi$$

と書ける.また連続時間の場合には

$$i\frac{d\ket \psi}{dt} = H \ket \psi$$

と書ける.この方程式の解は

$$U(t_1, t_2) = \exp{\frac{-iH(t_2-t_1)}{\hbar}}$$

であり,$U$はユニタリである.

2.2.3 Quantum measurement

閉じた系の量子力学的時間発展はユニタリ作用素で記述できるが,中で何が起きているのか観測するとその系は開く. 観測を定式化することを考える.

公準3

量子観測は${M_m}$という観測作用素の集合によって表現される. $M_m$は状態空間に作用する. $m$は観測で得られる結果で,$\ket \psi$を観測して$m$が得られる確率は

$$p(m) = \bra{\psi} M^\dagger _m M_m \ket {\psi}$$

であり,観測後の状態空間は

$$\frac{M_m \ket {\psi}}{\sqrt{\bra{\psi} M^\dagger _m M_m \ket {\psi}}} \tag{2.93}$$

となる.

$$\sum_m M^\dagger_m M_m = I \tag{2.94}$$

さらに

$$1 = \sum_m p(m) = \sum_m \bra{\psi} M^\dagger _m M_m \ket {\psi} \tag{2.95}$$

である.

単純かつ重要な観測の例にqubitのcomputational basisにおける観測がある.

$M_0 = \ket 0 \bra 0, M_1 = \ket 1 \bra 1$とする.

$$M_0 = (1, 0)^T (1, 0) = \left(\begin{array}{} 1 & 0 \ 0 & 0 \end{array}\right)=M_0^\dagger = M_0^2 $$

$M_1$も同様で,$I = M_0 + M_1 = \sum_m M^\dagger_m M$がたしかに成立する.

$\ket \psi = a\ket 0 + b \ket 1$であるとき,0, 1が観測される確率はそれぞれ

$$p(0) = (a*, b*) \left(\begin{array}{} 1 & 0 \ 0 & 0 \end{array} \right) (a, b)^T = (a*, b*) (a, 0)^T = |a|^2$$

$$p(1) = |b|^2$$

さらに観測後の状態は

$$\frac{M_0 \ket \psi}{|a|} = \frac{a}{|a|} \ket 0$$

$$\frac{M_0 \ket \psi}{|b|} = \frac{b}{|b|} \ket 1$$

となる. Sect. 2.2.7で$a/|a|$のような係数は実質無視でき,Chap. 1でみたように$\ket 0$か$\ket 1$の一方に確定する. (本では明示されていないが,2.93は観測によってある特定の$m$が得られたあとの状態空間のことなのだろう.)

2.2.4 Distinguishing quantum states

量子の世界では直交していない２つの状態を区別することが確率的にしかできない.

proof (Box 2.3)

ある観測$j$を得たときに状態が$i$であるというふうに推定する演算子$f: j \mapsto i$を考える.

背理法を用いる. 非直交な$\ket \psi_1, \ket \psi_2$を区別できると仮定する. $\ket \psi_1$があるとき,$f(j)=1$であるような$j$を観測する確率(jが2つ以上あれば総和は)は1でなければならない.

$E_i = \sum_{j: f(j)=1} M^\dagger_j M_j$とすれば

$$\bra \psi_1 E_1 \ket \psi_1 = 1, \bra \psi_2 E_2 \ket \psi_2 = 1 \tag{2.99}$$

である(行列演算の線形性).

(2.94)より$\sum_i E_i = I$で. 行列演算の線形性から$\sum_i \bra \psi_i E_i \ket \psi_i = 1$.

一方$\bra \psi_1 E_1 \ket \psi_1 = 1$だから$\bra \psi_2 E_2 \ket \psi_2 = 0$. 故に$\sqrt{E_2} \ket \psi_1 = 0.

$\ket\psi_2$に正規直行する$\ket \phi$と$\ket \psi_1$によって$\ket \psi_2 = \alpha \ket \psi_1 + \beta \ket \phi$と分解する.

$\ket \psi_1, \ket \psi_2$が正規直交でないなら,$|\beta| <1$である.このとき

$\sqrt{E_2} \ket \psi_2 = \beta \sqrt{E_2} \ket \phi$であって,

$$\bra \psi_2 E_2 \ket \psi_2 = |\beta|2 \bra \phi E_2 \ket \phi \leq |\beta^2| < 1 \tag{2.100}$$

で(2.99)と矛盾.　Q.E.D.
