#Quantum Information

量子情報3日目 (Nielsen and Chuang, Quantum Computation and Quantum Information)

$$ \def\bra#1{\mathinner{\left\langle{#1}\right|}} \def\ket#1{\mathinner{\left|{#1}\right\rangle}} \def\braket#1#2{\mathinner{\left\langle{#1}\middle|#2\right\rangle}} $$

注意:

$\ket \phi \otimes \ket \psi$はテンソル積. 本では単に$\ket \phi \ket \psi$とも書く(p.62 Figure 2.1)

1.3. 5 Qubit copying circut?

あるqubit $\ket \psi = a \ket 0 + b \ket 1$があるとして,それを２つ並べたqubitを作れるか試す.

古典的なビット($(a, b) = (1, 0) or (0, 1)$)であれば,CNOT($\ket {\psi 0}$)とすれば$\ket {00} \mapsto \ket{00}, \ket {10} \mapsto \ket{11}$より,確かにコピーが実現できている. しかし量子ビットであるとき,

$\ket \psi \ket 0 = (a, 0, b, 0)^T$. CNOTを作用させると

$CNOT (\ket \psi \ket 0) = (a, 0, 0, b)^T = a \ket{00} + b \ket{11}$

一方$\ket \psi \ket \psi = a^2 \ket {00} + ab \ket{01} + ba \ket{10} + b^2 \ket{11}$

が成立であり,等しくない.

より一般に,このようにqubitをコピーする操作は不可能であることが知られている(no-cloning theorem).

proof (p.532, Box 12.1)

ある量子コンピュータにスロットAとスロットBがあって,A,Bの状態はそれぞれ$\ket \psi, \ket s$とする. すなわちこのコンピュータの初期状態は$\ket \psi \otimes \ket s$である.

ユニタリな過程$U$によって,スロットAからBへのコピー

$$\ket \psi \otimes \ket s \mapsto U(\ket \psi \otimes \ket s) = \ket \psi \otimes \ket \psi$$

が実現できたとする.

これを$\ket \psi, \ket \phi$に行うと,

$$U(\ket \psi \otimes \ket s) = \ket \psi \otimes \ket \psi$$

$$U(\ket \phi \otimes \ket s) = \ket \phi \otimes \ket \phi$$

両辺の内積をとると

$\braket{\psi}{\phi} = (\braket{\psi}{\phi})^2$ (ホント？)

となり,直行する$\ket \phi, \ket \psi$でしか成立しない. q.e.d.
