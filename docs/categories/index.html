Quantum Neural Network 4.1 Representing the input 古典的なニューラルネットワークは典型的には不可逆な操作(ReLUとかmax poolingとか)だが,すでに見たように量子コンピュータは可逆な操作しか許さないので,量子ニューラルネットワーク(QNN)は可逆であり,入力$x_1, …, x_d$があって$y$を出力するときにはanciallary bitを挿入して $(x_1, …, x_d, 0) \rightarrow (x_1’, x_2’, …, x_d’, y)$という形になる(0がancillary bit) . $x_1. ,,, x_d$が古典びっとであるなら量子状態$\ket \psi _{1, …, d}$に変換するので,量子の世界での操作は$\ket \psi {1. ,,d} \ket 0 \rightarrow \ket {\psi’}{1, …, d} \ket y$というユニタリな操作となる. $\ket \psi _{1, …, d}$は$2^d$個の計算基底の重ね合わせと考えることができて,つまり$\ket \psi _{1, …, d} \in \mathcal{H}^{2^d} = \mathcal{H}^2 \otimes \cdots \otimes \mathcal{H}^2$, ここで$\mathcal{H}^2$は2-dim Hilbert spaceで基底は${\ket 0, \ket 1}$である.
